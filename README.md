# QueKMITL
## ขั้นตอนติดตั้ง
1. python -m pip install django <br>
2. python manage.py createsuperuser  (เพื่อสร้างพนักงานให้บริการ)<br>
3. python manage.py makemigrations <br>
4. python manage.py migrate <br>
5. python manage.py runserver <br>
